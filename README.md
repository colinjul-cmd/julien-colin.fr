# Julien Colin — Monteur

Site portfolio officiel de Julien Colin, monteur de films.

## À propos

Monteur expérimenté basé dans les Cévennes près de Montpellier.  
Spécialisé dans les **documentaires de création**, **fictions singulières** et **formes expérimentales**.

Trois longs métrages sortis en salle.

- **I&I** (2018) — Documentaire de Julie Hamiti (montage)
- **Le silence des nanos** (2007) — Cyberdocumentaire (réalisation + montage)
